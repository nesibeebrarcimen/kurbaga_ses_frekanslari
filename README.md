## Veri Seti (Dataset)

Bu projede kurbağa (Anuran) türlerinin ses kayıtlarından elde edilen akustik özellikleri (MFCC) içeren veri seti kullanılmıştır. Veri setinin kullanımı aşağıdaki lisans ve atıf kurallarına uygundur:

* **Veri Seti:** Anuran Calls (MFCCs)
* **DOI:** [10.24432/C5CC9H]
* **Yazarlar:** Colonna, J., Nakamura, E., Cristo, M., & Gordo, M. (2015)
* **Kaynak:** [UCI Machine Learning Repository](https://archive.ics.uci.edu/dataset/406/anuran+calls+mfccs)
* **Lisans:** [Creative Commons Attribution 4.0 International (CC BY 4.0)](https://creativecommons.org/licenses/by/4.0/legalcode)

Ekip projesidir.
